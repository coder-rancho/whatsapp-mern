# whatsapp-mern
a whatsapp clone using mern stack

### Technologies
  + **Express** : server
  + **mongoDB** : database
  + **mongoose** : Easily access database (messeges)
  + **Pusher** : Make database realtime. 
  + **socket.io** : implement two way connection to client-server
  + **firebase** : authentication and hosting
